Install create-react-app globally npm i -g create-react-app

Then create-react-app $APP_NAME

Then cd $APP_NAME

For http request, npm i axios

To install bootstrap and all kinds of icons: npm i bootstrap react-icons

In index.js file paste in this: import "bootstrap/dist/css/bootstrap.css"

Using react-icons, in App.js paste import { IconContext } from "react-icons"; at the top

Use it like this:

<IconContext.Provider value={{ color: "blue", size: "2em", style: { margin: "0 1em" } }} > <div> <Movies /> </div> </IconContext.Provider>

When everything is ready, npm start

Customizing Bootstrap with SCSS

  • npm i node-sass
  • In the src folder create a new file custom.scss

    Inside

    // Your variable overrides

    $primary: rgb(51, 255, 0);

    // Bootstrap and its default variables

    @import "../node_modules/bootstrap/scss/bootstrap";

  • Importing is very important!

    In the index.js file import the custom.scss under the bootstrap import.

    eg.import "bootstrap/dist/css/bootstrap.css";

    import "./custom.scss";
